# @anireact/typescript

> TypeScript config.

```
yarn add -D @anireact/typescript
```

`tsconfig.json`:

```json
{
    "compilerOptions": { "outDir": "dist", "rootDir": "src" },
    "extends": "@anireact/typescript"
}
```

## Features

-   Composite projects support.
-   Incremental build support.
-   Declarations only.
-   Strict mode.
-   No unused locals/parameters.
